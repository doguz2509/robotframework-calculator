# Robotframework Calculator

Extend RF for couple math operations with different data formats

## Math operation:
- Compare
- Add
- Subscript
- Multiple
- Divide

## Data formats: ##
- Robot time strings        - 2h 34m, ...
- Percents                  - 234 + 20%
- Bitrate (support iperf)  - 1G, 2g, 45m
